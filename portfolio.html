This is a group project I made for school during a 5 days workshop. It is a Survivors-like game in which you need to survive in an electronics shop for 20 minutes using all sorts of whacky weapons.

We worked on this project under the supervision of some members of Belgian game studio "Fishing Cactus".

My role was to create the player character and the weapons' behaviors using Blueprint. The player doesn't have a life bar. Instead every time they take damage, their flashlight battery power diminishes. If the flashlight turns off, it's game over! Each weapon is on auto fire and can be upgraded up to 9 times every time the player levels up. Some weapons couldn't be added in time unfortunately (such as the plunger).

A game about making tricks with a boat cruising through a sea of stars. Now with a real life steering helm!

This is a game made for school with my friends Luca Mevis, Cyril Naessens and Marc Verdonk during a 5 day workshop. We had to create an arcade game using an handmade custom controller. We decided to make a game in which you pilot a ship that can jump on waves and do tricks in the air, SSX style. All of that using a steering wheel made of papier-mache and a mouse. The game is made using Unreal Engine 5 (Blueprint). I loved working on this project, since it was using an alternative controller. Programming the player's behaviour was a totally different thing since I had to take the weird controller into account.

A project I made for school after a 5 days workshop with Unity. It is a 2D platformer in which you play as a little guy with a big shotgun.

I learned a lot on colisions and hit detection using raycast with this project. I also learned how to handle 2D projectiles and how to use coroutines. The thing I like the most about this project is the shotgun jump mechanic.

Once again, the assets in this project are not mine. They all come from the Unity Asset Store.

A web based strategy game in which you have to upgrade and buy ships to become the greatest space pirate!

This is a game I made with my friends Luca Mevis and Cyril Naessens for school. We had to create a game design document and then try to make a game out of it. I took care of developping the entirety of the game's backend. Since this is a multiplayer game, it uses Firebase's Realtime Database. This project helped me understanding how to create and use a database and how to organize it properly. It also helped me getting more familiar with Javascript since I had never done anything with it beforehand. The game is available here.
