A project I made for school after a 5 days workshop with Unity. It is a 2D platformer in which you play as a little guy with a big shotgun.

I learned a lot on colisions and hit detection using raycast with this project. I also learned how to handle 2D projectiles and how to use coroutines. The thing I like the most about this project is the shotgun jump mechanic.

Once again, the assets in this project are not mine. They all come from the Unity Asset Store.

This is a project I made with friends during Global Game Jam 2024. It is made using Unity. Over the course of the weekend, we managed to design and implement 11 mini-games exploiting all the possibilities offered by the controller.

I'm glad we could work on this game, as it allowed me to experiment things with alternative controllers, which is something I'd really like to dig in the future.

It is available on itch.io for free, although a Wii Guitar Hero guitar is needed to play it.
